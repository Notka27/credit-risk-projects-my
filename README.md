# Credit Risk Analysis — прогнозирование дефолта заемщиков
Проект по кредитному скорингу: предсказание вероятности невозврата кредита на основе данных о заемщике.
##  Цель проекта
Построить и сравнить модели машинного обучения для прогнозирования дефолта (loan_status = 1). Выявить ключевые факторы, влияющие на риск невозврата кредита.
##  Данные
- **Источник:** Kaggle — Credit Risk Dataset
- **Объем:** 32 581 запись, 12 признаков
- **Целевая переменная:** `loan_status` (0 — платежеспособен, 1 — дефолт)
### Признаки
| Признак | Описание |
|---------|----------|
| person_age | Возраст заемщика |
| person_income | Годовой доход |
| person_home_ownership | Тип жилья (аренда, собственность, ипотека) |
| person_emp_length | Трудовой стаж (лет) |
| loan_intent | Цель кредита |
| loan_grade | Кредитный класс (A-G) |
| loan_amnt | Сумма кредита |
| loan_int_rate | Процентная ставка |
| loan_percent_income | Отношение суммы кредита к доходу |
| cb_person_default_on_file | Наличие дефолта в прошлой кредитной истории |
| cb_person_cred_hist_length | Длина кредитной истории (лет) |
##  Исследовательский анализ данных (EDA)
### Ключевые выводы
- **Дефолт чаще встречается у заемщиков 20–30 лет**
- **Риск резко возрастает** при:
  - процентной ставке > 15%
  - отношении кредита к доходу > 0.3
  - кредитной истории < 5 лет
  - трудовом стаже < 10 лет
- **Безопасные категории:**
  - жилье в собственности (lowest risk)
  - цель кредита — бизнес (lowest default rate)
  - кредитный класс A-B
- **Высокий риск:**
  - аренда жилья (higher default)
  - цели: объединение долгов, ремонт, медицина
- **Дисбаланс классов:** 22% дефолтов, 78% платежеспособных
### Корреляционный анализ
| Признаки | Корреляция |
|----------|-----------|
| Возраст / длина кредитной истории | 0.86 (мультиколлинеарность, но для деревьев не критично) |
| Сумма кредита / отношение кредита к доходу | 0.57 |
| Остальные признаки | -0.25..0.38 |
## Предобработка данных
1. **Обработка выбросов** — ограничение 99-м процентилем (удалено ~16% данных)
   - Возраст > 80 лет
   - Доход > 99-го процентиля
   - Стаж > 99-го процентиля
   - Сумма кредита > 99-го процентиля
   - Процентная ставка > 99-го процентиля
   - Отношение кредита/доход > 99-го процентиля
2. **Заполнение пропусков** — медианой (`person_emp_length`, `loan_int_rate`)
3. **Кодирование категориальных признаков** — `LabelEncoder`
4. **Масштабирование числовых признаков** — `StandardScaler`
5. **Разделение данных** — train 80% / test 20% (стратификация по целевому признаку)
##  Модели
| Модель | AUC-ROC | Точность (Accuracy) | Recall (дефолт) | Precision (дефолт) |
|--------|---------|--------------------|-----------------|-------------------|
| Логистическая регрессия | 0.846 | 0.85 | 0.43 | 0.71 |
| **Random Forest** | **0.933** | **0.93** | **0.69** | **0.97** |
| Gradient Boosting | 0.929 | 0.92 | 0.67 | 0.93 |
### Лучшая модель — Random Forest
- **Точность предсказания дефолта (precision):** 97%
- **Обнаруживает реальных дефолтов (recall):** 69%
- **Ошибки:** 25 ложноположительных, 349 ложноотрицательных

## 📊 Результаты и интерпретация
### Ключевые факторы дефолта
- **Высокий кредитный класс (loan_grade)**
- **Низкий доход (или очень высокий)**
- **Цель кредита** — объединение долгов, ремонт, медицинские расходы
- **Наличие дефолта в прошлом**
- **Высокая процентная ставка (>15%)**
- **Высокое отношение кредита к доходу (>0.3)**
### Бизнес-применение
Модель может использоваться банками и МФО для:
- Автоматического одобрения/отказа в кредите
- Определения рискованных заемщиков
- Дифференциации процентных ставок
